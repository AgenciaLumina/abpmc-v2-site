✅ TUDO PRONTO! Apenas siga estes 3 passos:

───────────────────────────────────────────────────────────────
  PASSO 1: SALVAR O HTML
───────────────────────────────────────────────────────────────

1. Abra o terminal e execute:

   nano /tmp/socios-wordpress.html

2. Cole todo o HTML do WordPress que você forneceu

3. Salve com: CTRL+O (Enter) e CTRL+X

───────────────────────────────────────────────────────────────
  PASSO 2: EXECUTAR A IMPORTAÇÃO
───────────────────────────────────────────────────────────────

Execute no terminal:

   cd "/Volumes/Dock Station/abpmcdev/abpmc-v2"
   npx tsx scripts/importar-socios-html.ts

Aguarde a mensagem:

   ✅ IMPORTAÇÃO CONCLUÍDA!
   📊 Novos sócios: XXX
   🔄 Atualizados: XXX
   📈 Total processado: ~700

───────────────────────────────────────────────────────────────
  PASSO 3: TESTAR A PÁGINA
───────────────────────────────────────────────────────────────

1. Inicie o servidor (se não estiver rodando):

   npm run dev

2. Acesse no navegador:

   http://localhost:3000/socios

3. Teste:
   ✓ Buscar por nome
   ✓ Clicar nas abas alfabéticas
   ✓ Ver os cards dos sócios
